This project demonstrates how to manage complex state logic in React using the useReducer hook.
It implements a simple shopping cart where users can add products, remove them, and update quantities safely.

Features
Add products to cart

Remove products from cart

Increase and decrease product quantity

Prevent duplicate products in cart

Calculate total price dynamically

Strong error handling using optional chaining and nullish coalescing

Clean and predictable state management with useReducer

how to start the project:
git clone <repository-url>
In terminal : npm install
              npm start
